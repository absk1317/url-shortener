# README

* Ruby version 2.5.1
* Rails version 5.1.6

This URL shortener shortens a given URL and redirects to the original one when accessed by the shortened one.

---

This Shortener produces same results for same websites like:

```
google.com
www.google.com
http://google.com
https://google.com
http://www.google.com
https://www.google.com
```
